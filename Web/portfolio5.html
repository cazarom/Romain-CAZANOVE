configuration d'une carte Raspberry Pi

Ce projet réalisé en équipe contient un diaporama qui présente les étapes essentielles pour configurer une carte Raspberry Pi afin de l'utiliser comme un outil connecté et pilotable à distance. Nous y expliquons comment activer le protocole SSH pour contrôler la Raspberry Pi via un terminal sans clavier ni écran.Ce diapora y apporte aussi des détail sur la façon dont nous avont réalisé le câblage et la configuration du capteur de température, permettant de lire les données en temps réel directement dans le terminal.
